# bm-gp-EM
This contains the implementation and simulation code for Final Year Project on mixture models for stochastic processes:

**Chapter 2: Brownian motion mixture models**
- EM algorithm for Brownian motion mixtures (Section 2.3)
- Brownian motion simulations with EM results (Section 2.4)

**Chapter 3: Gaussian process mixture models**
- EM algorithm for Gaussian process mixtures using the Vecchia approximation (Section 3.4.4)
- GP simulations with EM results (Section 3.5)
- Sensitivity analysis of initialisation of GP EM algorthim (Section 3.6.1)
- Effects of different number of neighbours m in Vecchia approximation (Section 3.6.2)
